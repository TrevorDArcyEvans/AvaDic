# How to use AvaloniaDictionary


## Pre-requisites
*   .NET SDK 9


## Getting started
```bash
# clone repo
git clone https://github.com/TrevorDArcyEvans/AvaDic.git

# build
cd AvaDic
dotnet build

# run app
dotnet run
```
